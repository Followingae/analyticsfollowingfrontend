Getting Started

New to the platform? Start with one of these three actions.

Your Campaigns

Visit the Campaigns page to view active campaigns and track performance. Your account manager will set up campaigns for you via proposals.

/campaigns

Discover Creators

Search for Instagram creators on the Discover page. Unlock profiles to access AI-powered analytics.

/discover

Review Proposals

Check the Proposals tab in Campaigns for curated influencer lists from the Following team.

/campaigns
